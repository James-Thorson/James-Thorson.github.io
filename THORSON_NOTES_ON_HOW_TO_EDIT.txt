README:

Only need to edit:
_config.yml
index.html

MORE INFO: 
https://mmistakes.github.io/minimal-mistakes/docs/layouts/#splash-page-layout

EXPLANATION:
index.html -- is the main page
_layouts -- contains standard layouts referenced in "layout" header of other pages
_pages -- contains content and urls for linked pages
_posts -- contains blogposts for any page with layout="archive"
_data/navigation.yml -- contains banner links for pages within the site, which is displayed at top of each page.
